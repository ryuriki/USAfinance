# 株価可視化アプリ
有名米国株の株価をyahoo financeから取得してWebサイトに表示するコードです。

## 実行方法
コードが保管されているフォルダまで入る
```
% streamlit hello
% streamlit run main_kabu.py
```
Network URLをクリックすると表示できます、
